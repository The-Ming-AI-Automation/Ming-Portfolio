Featured Project — Restaurant AI Demo

Project Overview

This live, hosted demo (GitHub Pages) demonstrates how a small restaurant can use AI-driven automations to handle reservations, customer communication, and backend workflows — all integrated into one maintainable system.

Key Integrations

  • Tidio Chatbot — real-time engagement and conversational capture.
  • Vapi — AI voice agent designed to handle after-hours calls and automate customer support interactions.
  • Zapier — routes events to Google Calendar, Sheets, and email notifications.

Why this matters

The strength of this demo is that these tools coexist: Tidio handles conversational UI, Vipo ensures structured inputs, and Zapier reliably automates backend actions. That reduces manual work and improves customer experience.
