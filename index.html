Featured Project — Restaurant AI Demo

Project Overview

This live demo is a hosted restaurant website (GitHub Pages) that demonstrates practical automation: reservations, customer communication, and integrations tied together to form an operational assistant for small businesses.

Key Integrations

  • Tidio Chatbot — real-time customer engagement and AI-driven conversation flows that capture intent.
  • Vipo Integration — used to structure data capture and booking flows (e.g., reservation details, form data collection).
  • Zapier Automations — connects front-end events to backend actions (Google Calendar bookings, Sheets logging, email notifications).

Why this matters

The technical achievement is designing these tools to coexist rather than compete: Tidio handles conversational UI, Vipo structures/validates inputs, and Zapier reliably routes events to systems. The result is a single, maintainable workflow that reduces manual tasks and customer friction.
